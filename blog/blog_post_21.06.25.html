webOS - My project for Summer of Making

Today, I've begun developing my project for Summer of Making.

I decided to create a simulated operating system that runs in your webbrowser, mainly because I already did something similar in my Escape Room project.

This time though, I don't want to create a game, but concentrate more on realism of the simulation. I've had a lot of fun creating the Escape Room, but I enjoyed simulating Linux way more than creating a storyline or similar "game things".

Now, after spending the whole evening coding, I already wrote some parts: basic GUI features, terminal logic, filesystem integration, etc...

I'm really excited how this is going to turn out. It's my first hackathon and my first "coding competition" in general...
